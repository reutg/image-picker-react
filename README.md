# React Image picker

A simple React application where you can search for photos and save them to your favorites list.

**Demo**: https://reutg.github.io/image-picker-react

![screenshot](./assets/screenshot.png)

## Running the project

1. Clone the repo.
1. Run `npm install`.
1. Run `npm start`.

## Techstack

1.  React
1.  Material-UI
1.  MobX
